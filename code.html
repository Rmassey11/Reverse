The Code for Reverse


                    //Get the string from the page
//Controller Function
function getString(){
    document.getElementById("alert").classList.add("invisible");
    let userString = document.getElementById("userString").value;
    let revString = reverseString(userString);
    displayString(revString);
}

//Reverse the string
//Logic Function
function reverseString(userString){
    let revString = [];

    //reverse a string using a for loop
    for (let index = userString.length - 1; index >= 0; index--) {
        revString += userString[index];
        
    }
    return revString
}

//Display the reversed string to the user
//View Function
function displayString(revstring){
    //write message to the page
    document.getElementById("msg").innerHTML = `Your string reversed is: ${revstring}`;
    //show the alert box
    document.getElementById("alert").classList.remove("invisible");
}

The Code is structured in three functions.

getString

The getString function gets the users input from the page for use in our other functions. First we get our alert element and ensure that it starts as "invisible" until our app runs and displays the results. We then declare two Block Scoped variables userString and revString. userString is set to the value the user has input from our form on the page. revString is set to the value returned from our reverseString function. It also calls the function to run. Once reverseString returns a value we then call our displayString function with the returned value passed into it.

reverseString

The reverseString function takes the value gathered by getString and reverses it. First we declare a Block Scoped variable revString and set it to an empty array. Since strings are also arrays themselves we then run a "for loop" through the string array passed to us as userString from our getString function. The loop begins at the LAST value in the array are iterates to the FIRST. During each iteration the current value at each index is added to our revString array effectively reversing the original string. Finally revString is then returned to our getString function to then be passed into the displayString function.

displayString

The displayString function takes our revString from the reverseString function and displays it on our page through bootstraps alert functionality. First we call on our paragraph tag by its Id "msg" and using template literal we set our display message along with our revString variable. Finally we remove the "invisible" class from our alert so that the message is then displayed to the user in reverse from their original input.
